≪主な機能≫

　・カウントダウン
　・ホットキー機能

―――――――――――――――――――――――――――――――――――――
≪使い方≫

　　DisplayPowerOff.exeを起動して出てきたボタンを押すと
　　ディスプレイの電源を消すことができます。
　　
　　通知領域のアイコンをクリックすると
　　タスクトレイに格納することもできます。


　　※注意
　　　１．ホットキーの設定次第ではほかのアプリケーションのショートカットキーに影響を及ぼします
　　　　　
　　　　　例えば、ホットキーとして「Control + T」を設定した場合、
　　　　　このソフトウェアの起動中は、
　　　　　他のアプリケーション（ブラウザなど）で
　　　　　新しいタブが開けなくなるなどの現象が起こります。

　　　２．電源を消してもマウスを動かしたり、キーボードを押したりすると
　　　　　すぐに復帰するので注意が必要です。
